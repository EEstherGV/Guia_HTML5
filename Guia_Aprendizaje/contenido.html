¿Qué es HTML?


HTML (hypertext markup language), es utilizado para la creacion de paginas web. Funciona mediante un sistema de etiquetas que el sistema es capaz de interpretar como objetos.

Versiones de HTML


HTML 2.0

Creada en 1995. Fue la primera version de HTML, el cual estaba basado en SGML (Lenguaje marcado generalizado estandar)

HTML 3.2

Creada en 1997. Esta version contenia applets de java y texto alrededor de las imagenes.

HTML 4.01

Creado en 1999. Se centraba en el desarrollo de estandar XHTML.

HTML5 y sus derivados

Para dinales de octubre de 2014 se nombro HTML5 como el estandar oficial. Se introducen nuevas etiquetas, poder agregar audios y videos directamente de la web.



Editores de texto web


Es el programa utilizado para la creacion de una pagina web. Tambien son conocidos como editores de codigo o incluso son conocidos como editores de HTML, estos programas son utilizados para desarrollar las bases de una pagina web, ademas de incluir funcionalidades adicionales como lo son las correciones de errores y los autocompletables.

Existen una cantidad elevada de editores, sin embargo, los mas utilizados son:

Visual Studio Code

Herramienta multi codigo de edicion HTML. Es una herramienta bastante completa que trabaja de la mano con otros programas como lo es python, permitiendo una gran variedad de conexiones a nivel codigo con otros entornos.



Sublime text

Herramienta de edicion de texto basica, es utilizada tanto por principiantes como por progrmadores con experiencia, ya que su agradable y comoda sintaxis permite la facil adaptacion y comprencion. Sin embargo, posee algunas desventajas como la limitacion que esta trae a la hora de conectar programas y descargar plugins, debido a que este programa no cuenta con una estensa lista para esto.



Atom

Esta herramienta es reconocida como una de las mejores que hay hasta el momento, pues la extensa variedad de actividades que puedes realizar permiten al usuario tener una experiencia agradable y que con las mismas herramientas que facilita el programa dejaran un codigo bastante decente.



Etiquetas HTML

Las etiquetas en HTML son la forma en la que el sistema lee la instrucciones que el usuario busca ejecutar, con marcados de facil comprencion para las personas sin la necesidad de compilarlo primero. Estas etiquetas o instrucciones son la base del desarrollo del sitio web, ya que lo qe se busca es dar una orden a la maquina que ejecute la forma en la queremos que el texto se refleje y en que condiciones.

Las etiquetas mas comunes en HTML son:



Body

Así como su nombre lo indica el body es el cuerpo de todo el documentp HTML, este contiene todo el codigo a desarrollar e indica donde empieza el contenido visible.



Head

Una parte muy escencial de la estructura HTML es el head ya que aqui se retienen las relaciones de hojas, metadatos, etc. que permiten la funcionalidad del codigo y sus relaciones.



Strong

Esta es una etiqueta que permite poner palabras en tono de negrilla.



Div

Con esta etiqueta podemos crear contenedores para retener y reubicar informacion.



a

Aqui podemos crear enlaces que envien a otra direccion.



br

Permite hacer saltos de linea en un texto.



p

Es la etiqueta utilizada para referir un parrafo.



h1 y sus derivados

Las etiquetas de h permiten generar diferente niveles de titulos, dependiendo del numero este aumenta su tamaño o importancia y relevancia.



Img

Con esta etiqueta podremos llamar a una imagen ya sea desde un enlace o una imagen previamente guardada.



Link

Utilizada para enlazar recursos externos al trabajado.



Nav

Es un identificador que permite al sistema reconocerlo como parte del navegador.



footer

Este es el pie de pagina, contiene informacion extra referente a la pagina.



Aside

Es utilizado para nombrar ese contenido de la web que no es relevante y que puede ser adicional.



button

Es la propiedad que permite representar botones en los formularios.



Input

Crea un campo para recibir informacion de un usuario.



textarea

Permite al usuario crear textos grandes con cierto limite.



Table

Esta etiqueta inicia y cierra la creacion de una tabla.



Li

Genera listas desordenadas.



Th

Esta es una etiqueta que define el encabezado de una celda.



form

Permite la creacion de formularios.
